Description :
Dans ce jeu 2D, le joueur contrôle un personnage pour trouver des clés cachées et les ramasser afin d’ouvrir une porte et accéder au trésor.

Technologies :

- Java
- IDE : Eclipse

Fonctionnalités :

- Déplacement du personnage avec les touches fléchées
- Ramassage des clés et ouverture de la porte
- Objectif final : atteindre le trésor
